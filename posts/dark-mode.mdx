## Read the Docs

[Tailwind](https://tailwindcss.com/docs/dark-mode) has great support for dark mode out of the box. 

This site supports dark mode by first using the `class` strategy in the `tailwind.config.js` file:

```js
module.exports = {
  // ...
  darkMode: 'class',
}
```
